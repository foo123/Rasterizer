# Rasterizer

Rasterize, stroke and fill lines, rectangles, curves and paths. Even without canvas.

**version: 0.9.5** (13 kB minified)

**What is not supported yet

1. `round` lineCap
2. `round` lineJoin
3. lineDash (only solid lines)
